Módulo Administrativo

1. Como administrador quiero registrar y editar la información general de la empresa, como nombre comercial, razón social, cédula jurídica, teléfonos, correos y dirección para mantener actualizados los datos institucionales del sistema.

2. Como administrador quiero gestionar parámetros generales del sistema como tipos de impuestos.

3. Como administrador quiero visualizar la estructura general del sistema mostrando los usuarios, empleados y roles registrados para mantener control sobre la organización del sistema.

Módulo Roles						

4. Como administrador quiero un formulario para crear roles dentro del sistema para definir los distintos niveles de acceso de los usuarios.

5. Como administrador quiero visualizar y editar o desactivar roles existentes para ajustar la estructura de permisos del sistema.

6. Como administrador quiero asignar permisos a cada rol para controlar el acceso a los distintos módulos y funcionalidades del sistema.

Módulo Usuarios						

7. Como administrador quiero una vista que me permita visualizar, filtrar, registrar y editar usuarios con nombre, correo, rol y estado (activo/inactivo) para controlar quién accede al sistema.

8. Como usuario quiero un formulario para autenticarme mediante usuario y contraseña para acceder de forma segura a la plataforma.

9. Como usuario quiero recuperar mi contraseña mediante un formulario donde ingrese mi correo electrónico registrado para que el sistema envíe un enlace con un token de recuperación que permita establecer una nueva contraseña de forma segura.

Módulo Empleados

10. Como administrador quiero un formulario para registrar, editar y activar o desactivar empleados con datos como nombre completo, identificación, puesto, teléfono, correo, dirección, contacto de emergencia, alergia a medicamentos y estado (Activo/Inactivo) para mantener actualizada la información del personal de la empresa.

11. Como administrador quiero ver y consultar toda la información personal de los empleados, así como visualizar la distribución del personal por área o puesto para apoyar decisiones administrativas.

12. Como empleado quiero poder visualizar y editar mi información personal como padecimientos o enfermedades, contactos de emergencia, teléfono y correo.

Módulo Clientes

13. Como personal de ventas quiero un formulario para registrar clientes con datos como nombre, identificación, estado (Activo/Inactivo), teléfono, correo y dirección para mantener una base de datos actualizada.

14. Como personal de ventas necesito visualizar y editar la información de los clientes como nombre, estado (Activo/Inactivo), teléfono, correo y dirección, esto para mantener los datos correctos y transparentas para su análisis comercial. 

Módulo Proveedores

15. Como personal de proveeduría quiero registrar proveedores mediante un formulario donde se ingresen datos como nombre de la empresa, identificación, teléfono, correo electrónico, dirección y estado (activo/inactivo), para mantener un registro de las entidades que suministran productos al negocio.

16. Como personal de proveeduría quiero visualizar, filtrar por nombre, identificación o estado para agilizar el proceso y editar en caso de ser necesario la información de los proveedores mediante un formulario de actualización para corregir o modificar datos como teléfono, correo electrónico o dirección.

Módulo Compras

17. Como personal de proveeduría quiero un formulario para registrar órdenes de compra a proveedores indicando proveedor, productos, cantidades, precios y fecha para registrar las adquisiciones de inventario tambien filtrar por compras realizadas, proveedores o fecha para consultar historial.

18. Como personal de proveeduría quiero editar o cancelar órdenes de compra antes de ser confirmadas para corregir errores en la solicitud.

19. Como personal de proveeduría quiero confirmar una orden de compra para que el sistema registre automáticamente la entrada de los productos al inventario.

Módulo Inventario

20. Como personal de proveeduría quiero un formulario para ingresar y poder visualizar y editar productos del sistema con sus características como nombre, categoría, estado (Activo/Inactivo), precio de compra, precio de venta, tipo de impuesto y stock.

21. Como personal de ventas quiero consultar y filtrar el inventario por producto o categoría para verificar disponibilidad antes de registrar un pedido.

22. Como personal de proveeduría quiero que el sistema registre automáticamente las salidas de productos asociadas a la confirmación de pedidos para evitar errores manuales.

23. Como personal de proveeduría quiero recibir alertas cuando el stock de un producto sea bajo para evitar desabastecimiento.

24. Como personal de proveeduría quiero consultar y filtrar el historial de movimientos de inventario para mantener trazabilidad.

Módulo Pedidos

25. Como personal de ventas quiero un formulario para registrar, editar y confirmar órdenes de pedido con datos del cliente, productos solicitados, cantidades, fechas y estado (Confirmado/Cancelado/Entregado) para registrar las solicitudes de compra.

26. Como personal de ventas quiero cancelar pedidos cuando el cliente lo solicite para mantener actualizado el registro de órdenes.

27. Como administrador quiero visualizar los clientes más frecuentes y los productos más solicitados para apoyar la toma de decisiones comerciales.

28. Como personal de logística, quiero visualizar y filtrar los pedidos confirmados por fecha o estado para coordinar el despacho de los productos.

29. Como personal de ventas quiero que el sistema calcule automáticamente el subtotal, impuestos y total de cada pedido según los productos seleccionados.

30. Como personal de logística quiero visualizar la información del cliente, asociada a un pedido para coordinar entregas correctamente.

Módulo Licitaciones

31. Como administrador quiero un formulario para registrar, editar y visualizar concursos o solicitudes de cotización de clientes institucionales con datos como institución, persona de contacto, descripción del requerimiento, fechas clave y estado para iniciar su seguimiento y actualización de estado (en análisis, en preparación, oferta enviada, adjudicado, no adjudicado) para controlar su avance, se podrá filtrar para por institución, estado y fechas para una búsqueda precisa de las cotizaciones así como también su respectiva descarga en formato PDF o Excel.

32. Como administrador quiero adjuntar documentos relacionados con el proceso (solicitud recibida, cotizaciones, certificaciones, fichas técnicas) para centralizar la información.

33. Como administrador quiero un formulario para gestionar los productos, cantidades y precios ofertados en cada proceso para construir la propuesta de venta, permitiendo que el sistema calcule subtotales por producto y total general de la oferta.

34. Como administrador quiero recibir recordatorios de fechas clave registradas (envío de oferta, consultas o entrega) para evitar incumplimientos.

35. Como administrador quiero que el sistema sugiera y complete automáticamente información repetitiva de la empresa (datos legales, representante, plantillas de cotización) para agilizar la preparación de ofertas.

Módulo Finanzas

36. Como administrador quiero que el sistema registre automáticamente los movimientos financieros (ingresos y egresos) generados a partir de pedidos, pagos a proveedores y otros registros del sistema para mantener un control financiero actualizado, cuenta con opciones para consultar y filtrar los movimientos financieros por fecha, tipo o categoría.

37. Como administrador necesito ver los ingresos, egresos y balance general para evaluar el estado financiero del negocio.

38. Como administrador necesito registrar y gestionar pedidos por cobrar asociados a clientes, permitiendo actualizar su estado (pendiente, pagado o vencido) para dar seguimiento a los pagos pendientes.

39. Como administrador necesito registrar y gestionar las cuentas por pagar a proveedores y servicios, permitiendo actualizar su estado (pendiente, pagado o vencido) para controlar las salidas de dinero.

40. Como administrador necesito recibir alertas de pagos o cobros próximos a vencerse para evitar atrasos. 

Módulo Predicciones

41. Como administrador quiero que el sistema analice automáticamente el historial de pedidos para identificar patrones de compra de los clientes y los productos que adquieren con mayor frecuencia.

42. Como administrador quiero visualizar las predicciones de productos que un cliente podría requerir nuevamente según el análisis automático de su historial de compras para la planificación del inventario y en la toma de decisiones.


Módulo Reportes

43. Como administrador quiero visualizar un dashboard general con métricas de ventas, inventario y licitaciones para toma de decisiones.

44. Como administrador quiero generar reportes de inventario, pedidos y licitaciones para analizar el movimiento general y analisis estrategico.

45. Como administrador quiero exportar reportes en formato PDF o Excel para compartir información con otras áreas o realizar análisis externos.

46. Como administrador quiero consultar y filtrar reportes por período (semanal, quincenal y mensual) para dar seguimiento periódico al desempeño del negocio.

47. Como administrador quiero generar reportes de predicciones de compra por cliente o producto según un período de tiempo definido para analizar tendencias de demanda.
