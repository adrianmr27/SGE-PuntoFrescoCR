1. Como administrador quiero un formulario para registrar, editar y visualizar concursos o solicitudes de cotización de clientes institucionales con datos como institución, persona de contacto, descripción del requerimiento, fechas clave y estado para iniciar su seguimiento y actualización de estado (en análisis, en preparación, oferta enviada, adjudicado, no adjudicado) para controlar su avance, se podrá filtrar para por institución, estado y fechas para una búsqueda precisa de las cotizaciones así como también su respectiva descarga en formato PDF o Excel. (No cabe completa)

Escenario 1: Registrar licitación

En caso de nueva solicitud institucional cuando el administrador completa el formulario el sistema registra la licitación con estado inicial

Escenario 2: Actualizar estado

En caso de seguimiento activo cuando se cambia el estado del proceso el sistema guarda el nuevo estado

Escenario 3: Filtrar y exportar licitaciones

En caso de múltiples registros cuando se aplican filtros o se exporta el sistema muestra resultados y genera archivo

2. Como administrador quiero adjuntar documentos relacionados con el proceso (solicitud recibida, cotizaciones, certificaciones, fichas técnicas) para centralizar la información.

Escenario 1: Adjuntar documento

En caso de licitación existente cuando se carga un archivo el sistema lo almacena y asocia

Escenario 2: Visualizar documentos

En caso de archivos cargados cuando se accede a la licitación el sistema muestra los documentos

Escenario 3: Validar tipo de archivo

En caso de formato no permitido cuando se intenta subir el sistema rechaza el archivo

3. Como administrador quiero un formulario para gestionar los productos, cantidades y precios ofertados en cada proceso para construir la propuesta de venta, permitiendo que el sistema calcule subtotales por producto y total general de la oferta.

Escenario 1: Agregar productos a oferta

En caso de licitación activa cuando se ingresan productos el sistema los añade a la propuesta

Escenario 2: Calcular subtotales

En caso de productos registrados cuando se definen cantidades y precios el sistema calcula subtotales

Escenario 3: Calcular total oferta

En caso de propuesta completa cuando se visualiza resumen el sistema muestra total general

4. Como administrador quiero recibir recordatorios de fechas clave registradas (envío de oferta, consultas o entrega) para evitar incumplimientos.

Escenario 1: Detectar fechas próximas

En caso de fechas registradas cuando se acerca la fecha límite el sistema genera recordatorio

Escenario 2: Notificar usuario

En caso de recordatorio activo cuando el usuario accede el sistema muestra alerta

Escenario 3: Evitar duplicados

En caso de recordatorio ya enviado cuando persiste la condición el sistema no repite la alerta

5. Como administrador quiero que el sistema sugiera y complete automáticamente información repetitiva de la empresa (datos legales, representante, plantillas de cotización) para agilizar la preparación de ofertas.

Escenario 1: Autocompletar datos

En caso de datos previamente registrados cuando se crea una oferta el sistema completa información

Escenario 2: Sugerir plantillas

En caso de licitaciones similares cuando se inicia propuesta el sistema sugiere plantillas

Escenario 3: Permitir edición

En caso de datos autocompletados cuando se modifican el sistema guarda cambios
