1. Como administrador quiero que el sistema analice automáticamente el historial de pedidos para identificar patrones de compra de los clientes y los productos que adquieren con mayor frecuencia.

Escenario 1: Analizar historial

En caso de pedidos registrados cuando se ejecuta análisis el sistema identifica patrones

Escenario 2: Detectar productos frecuentes

En caso de datos suficientes cuando se procesa información el sistema identifica productos

Escenario 3: Actualizar análisis

En caso de nuevos pedidos ando se registran el sistema recalcula patrones

2. Como administrador quiero visualizar las predicciones de productos que un cliente podría requerir nuevamente según el análisis automático de su historial de compras para la planificación del inventario y en la toma de decisiones.

Escenario 1: Mostrar predicciones

En caso de análisis previo cuando se consulta cliente el sistema muestra sugerencias

Escenario 2: Filtrar predicciones

En caso de múltiples clientes cuando se selecciona cliente el sistema muestra resultados

Escenario 3: Basar en historial

En caso de datos históricos cuando se generan predicciones el sistema usa compras previas
