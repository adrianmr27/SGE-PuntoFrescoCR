1. Como personal de ventas quiero un formulario para registrar clientes con datos como nombre, identificación, estado (Activo/Inactivo), teléfono, correo y dirección para mantener una base de datos actualizada.

Escenario 1: Registrar cliente

En caso que el usuario acceda al formulario de clientes cuando ingrese los datos del cliente y presione guardar el sistema registra el cliente

Escenario 2: Validar datos requeridos

En caso que el usuario complete el formulario de cliente cuando deje campos obligatorios vacíos el sistema muestra mensajes de validación

Escenario 3: Confirmar registro exitoso

En caso que el cliente sea registrado correctamente cuando el sistema finaliza el registro el sistema muestra confirmación de registro exitoso

2. Como personal de ventas necesito visualizar y editar la información de los clientes como nombre, estado (Activo/Inactivo), teléfono, correo y dirección, esto para mantener los datos correctos y transparentas para su análisis comercial.

Escenario 1: Consultar cliente

En caso que existan clientes registrados cuando el usuario busque o seleccione un cliente el sistema muestra la información del cliente

Escenario 2: Editar información del cliente

En caso que el usuario visualice un cliente cuando modifique sus datos el sistema actualiza datos

Escenario 3: Filtrar clientes

En caso de que los clientes sean listados cuando se aplican filtros el sistema muestra filtrados
