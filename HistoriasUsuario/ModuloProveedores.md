1. Como personal de proveeduría quiero registrar proveedores mediante un formulario donde se ingresen datos como nombre de la empresa, identificación, teléfono, correo electrónico, dirección y estado (activo/inactivo), para mantener un registro de las entidades que suministran productos al negocio. (No cabía completa)

Escenario 1: Registrar proveedor

En caso de registrar un proveedor nuevo cuando ingrese los datos del proveedor y presione guardar el sistema registra proveedor

Escenario 2: Validar datos

En caso de datos incompletos al registrar un proveedor cuando intenta guardar el sistema muestra errores

Escenario 3: Evitar duplicados

En caso de proveedor existente cuando se registra el sistema bloquea duplicados

2. Como personal de proveeduría quiero visualizar, filtrar por nombre, identificación o estado para agilizar el proceso y editar en caso de ser necesario la información de los proveedores mediante un formulario de actualización para corregir o modificar datos como teléfono, correo electrónico o dirección. (No cabía completa)
Escenario 1: Consultar proveedores

En caso de proveedores registrados cuando el usuario busque o seleccione un proveedor el sistema muestra la información del proveedor

Escenario 2: Editar proveedor

En caso de cambios en un proveedor cuando se modifica el sistema actualiza los datos

Escenario 3: Filtrar proveedores

En caso de múltiples registros cuando se aplican filtros el sistema muestra el resultado filtrado
